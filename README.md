# GameD - Python Game Server Manager

---

## Overview

**GameD** is a Python project for managing game servers.  
It provides a framework to start, stop, and manage servers programmatically, with support for both Windows and Linux.  

The project is designed to simplify server setup, handle environment requirements, and organize server files automatically.

---

## Status

⚠️ This project is currently under development. Features are evolving, and some functionality may change over time.
